# CodeAlpha Internship Task 2: Exploratory Data Analysis (EDA)

## Project Overview

This project was completed as part of the CodeAlpha Data Analytics Internship Program. The objective was to perform Exploratory Data Analysis (EDA) on a book dataset and derive meaningful insights through statistical analysis and data exploration techniques.

## Objective

To analyze the book dataset, identify trends and patterns, detect anomalies, and gain insights into book prices and ratings using Exploratory Data Analysis.

## Dataset Information

The dataset contains information about books collected through web scraping.

### Features

- Book Title
- Price (£)
- Rating

## Technologies & Libraries Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Analysis Performed

The following analysis tasks were carried out:

1. Data loading and inspection
2. Data cleaning and preprocessing
3. Statistical summary generation
4. Rating distribution analysis
5. Price distribution analysis
6. Identification of highest and lowest priced books
7. Relationship analysis between price and rating
8. Detection of unusual price values (outliers)

## Questions Explored

- What is the average price of books?
- Which rating category appears most frequently?
- What are the highest and lowest priced books?
- Are highly rated books more expensive?
- Are there any unusual price values in the dataset?

## Key Findings

- The average book price is approximately **£38**.
- One-star books are the most common rating category.
- Book prices vary significantly across different titles.
- A few books have unusually high prices compared to the dataset average.
- No strong relationship was observed between book ratings and prices.
